# IQA_Software
基于PEID数据库的主观和客观质量评价软件  
  
### 此项目共包括两个软件  
* 主观评价软件（PyQt5_Photo_Display）  
* 客观评价软件（Object_IQA_Software）  
----------------------------------------------
### 主观实验软件  
  
#### 功能  
* 开展图像的主观质量评价实验；
* 目前包括开始，实验背景介绍，实验预热，主实验台；
* 支持DSIS、SS、DR+Staircase、单双屏显示等实验；
* 通过设置面板可以对实验可以进行一定程度的参数自定义。  

----------------------------------------------  
### 客观实验软件  
  
#### 功能  
* 开展图像客观算法评分；
* 目前支持一些常见的NR和FR算法及其批处理。
